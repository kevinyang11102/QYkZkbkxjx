# 核心代码

以下是项目中的一段核心代码示例，展示了如何使用Spring Boot框架进行商品列表查询：

```java
// 商品服务类
@Service
public class ProductService {

    // 注入商品Repository
    @Autowired
    private ProductRepository productRepository;

    // 查询商品列表
    public List<Product> listProducts() {
        return productRepository.findAll();
    }
}
```
